# Wesley Feng 作品集网站

这是一个可以直接打开的静态作品集网站，主要展示：

- AI 视频作品
- 抖音 AI 视觉案例
- 视频剪辑、AIGC、视觉包装相关能力
- PDF 简历下载

## 怎么查看

双击 `index.html` 即可在浏览器中打开。

## 当前公开资源

- 页面入口：`index.html`
- 样式：`assets/css/styles.css`
- 交互脚本：`assets/js/script.js`
- 图片素材：`assets/images/`
- 视频素材：`assets/videos/glide-promo.mp4`
- PDF 简历：`assets/pdf/冯文泽_简历.pdf`

## 目录说明

- `assets`：网站实际使用的图片、视频、PDF、CSS、JS。
- `docs`：项目记录和阶段总结。
- `tools`：生成或处理作品素材的辅助脚本。
- `00_面试准备`：面试准备文档。
- `01_视频作品与素材`：视频作品和素材整理区。
- `02_系统报告与备份`：系统检查报告、Chrome 修复备份等。
- `03_旧实验与归档`：旧实验、历史文件、旧版简历和源文件。

## 注意

网站公开下载的简历只保留 `assets/pdf/冯文泽_简历.pdf`。旧版 PDF、源文件和临时预览已经移动到归档目录，避免下载链接混乱。
