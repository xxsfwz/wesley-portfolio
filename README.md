# 数字媒体作品集网站

这是一个可以直接打开的静态作品集网站，适合先作为作品集雏形，后续逐步替换成你的真实项目。

## 文件结构

```text
New project/
  index.html
  README.md
  assets/
    css/
      styles.css
    js/
      script.js
    images/
    videos/
    pdf/
```

## 怎么查看

双击 `index.html`，就能在浏览器里打开。

## 后续怎么替换

- 修改个人介绍：打开 `index.html`，搜索“关于我”
- 修改作品分类文字：搜索“作品分类”
- 修改精选项目：搜索“精选项目占位”
- 修改项目示例：搜索“未来城市交互展厅”
- 修改后续加分项：搜索“后续加分项”
- 修改制作流程：搜索“AI辅助创作流程”
- 修改简历区域：搜索“简历预留区”
- 修改联系方式：搜索“联系我”
- 调整颜色和版式：修改 `assets/css/styles.css`
- 当前版本使用轻量微交互；如需调整噪点转场、滚动出现和 hover 提示，可修改 `assets/js/script.js`

## 素材放哪里

- 图片、作品封面、过程图：放进 `assets/images/`
- 视频、Showreel、项目录屏：放进 `assets/videos/`
- PDF 简历、证书扫描件：放进 `assets/pdf/`

## 建议下一步准备的素材

- 3D作品渲染图：每个项目 3 到 6 张
- AE/PR作品：视频封面图、15 到 60 秒短视频
- Unity作品：运行截图、录屏、项目说明
- 每个项目一段 80 到 150 字简介
- 项目过程图：草图、建模过程、AE时间线、Unity场景截图
- 简历素材：证书照片、获奖名称、项目经历、社团或工作经历、PDF简历

## 作品集建议路线

1. 先保留当前模板结构，把个人信息、联系方式和简历内容补完整。
2. 选择 3 个最能代表你的方向的作品，分别放入动态影像、3D/Unity、AI工具力/网页实验类别。
3. 每个作品都准备“最终效果 + 制作过程 + 我的职责 + 使用工具”。
4. 后续再增加独立项目详情页、Showreel 视频区和 PDF 简历下载。
